Real-time availability (ex. Microsoft Teams) to see who is currently in office, remote, or offline.

User story: 
James is on a team of 11 computer science students in a software engineering class. They each have busy schedules and want to maximize their productivity by picking the best times to meet. They have used When2Meet but feel the UI is clunky and lacks features. It would be nice to have some of the above features in one place.

User Persona:
Morpheus is an expert software engineer and utilizes tools such as Zoom, Slack, GitHub Actions, etc. so he knows what features he needs in collaboration tools. If there were other software that had attractive features packaged with a clean UI he would consider switching.
